# IQ Arena — Ranked Duel Rules

## Базовый режим

- 2 игрока
- 5 раундов
- 1 вопрос на раунд
- 4 варианта ответа
- 1 правильный ответ
- 10 секунд на ответ
- сервер открывает вопрос одновременно для двух игроков
- сервер принимает только первый ответ игрока
- сервер считает очки и рейтинг

## Очки

- неправильный ответ: 0
- таймаут: 0
- правильный ответ: 100 + бонус скорости
- быстрый ответ должен давать преимущество, но не ломать баланс

## Античит

- вопрос не отправляется заранее
- correctIndex не отправляется до reveal
- ответ принимается только сервером
- клиент не решает победителя
