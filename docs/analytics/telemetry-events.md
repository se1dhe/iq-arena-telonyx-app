# Telemetry Events

Telemetry should be minimal and gameplay-oriented.

## MVP Events

- `auth_login_success`
- `ws_connected`
- `queue_joined`
- `queue_left`
- `match_found`
- `round_opened`
- `answer_submitted`
- `answer_rejected_late`
- `round_scored`
- `match_completed`
- `rating_updated`
- `question_reported`
